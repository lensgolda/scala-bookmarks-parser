# bookmarks-parser

Exported (json) bookmarks parser written in scala, with Cats :cat:

An example application built from the [ce3.g8 template](https://github.com/typelevel/ce3.g8).

```toml
[tools]
sbt = "latest"
scala = "2.13.15"
scala-cli = "latest"
```

## Run application

```shell
sbt run
```

## Run tests

```shell
sbt test
```
